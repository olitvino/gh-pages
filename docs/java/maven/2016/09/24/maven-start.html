Maven start

Create project 

$ mvn archetype:generate -DgroupId=ua.org.jee.tutorial \
  -DartifactId=simple \
  -Dpackage=ua.org.jee.tutorial \
  -Dversion=1.0-SNAPSHOT

Build project 

$ cd simple
$ mvn install

Run project 

$ java -cp target/simple-1.0-SNAPSHOT.jar ua.org.jee.tutorial.App
Hello World!

Effective pom 

$ mvn help:effective-pom
